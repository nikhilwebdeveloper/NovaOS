# NovaOS Design Principles

## Philosophy

NovaOS is designed to be lightweight, modern and fast.

Every design decision must improve one or more of these:

- Performance
- Simplicity
- Accessibility
- Productivity
- Consistency

If a feature does not improve the user experience, it should not exist.

---

## Core Values

- Performance First
- Minimal but Powerful
- Clean User Interface
- Low Resource Usage
- Consistent Design
- No Bloat
- User Control
- Hardware Compatibility

---

## Tagline

Power Without Weight.
